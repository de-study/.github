# Data Engineer Book Review Study

데이터 엔지니어링 관련 도서를 함께 읽고, 책에서 다루는 개념과 인사이트를 정리하여 공유하는 스터디입니다.

<br>

## 스터디 소개 및 목적

본 스터디는 데이터 엔지니어링 분야의 기술서를 정기적으로 함께 읽고, 각자의 관점에서 정리한 리뷰를 모으기 위해 만들어졌습니다.

운영 목적은 다음과 같습니다.

- 분량이 많아 개인 학습으로는 완독이 쉽지 않은 기술서를 정해진 일정 안에서 끝까지 함께 읽어 나갑니다.
- 평소 관심은 있었지만 시작할 계기가 부족했던 데이터 엔지니어링 주제를 스터디를 계기로 본격적으로 다룹니다.
- 같은 책이라도 각자의 경험과 도메인에 따라 해석이 달라지는 부분을 토론으로 보완하고, 혼자서는 닿지 못했을 관점까지 학습 범위를 확장합니다.
